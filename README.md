# Запук и пара полезностей

1. Создай вирт окружение - python -m venv venv
2. Активируй - venv\Scripts\activate
3. Установка Зависимостей - pip install -r requirements.txt
3.3 Не забывай обновлять зависимости в файле - pip freeze > requirements.txt

4. Запуск проекта uvicorn app.main:app --reload

# Цели код на fastapi бд sqllite
Пользователи
    - Регистрация с email и паролем
    - Авторизация / выход
    - Роли: обычный, менеджер, админ
    - Обновление профиля
    - Удаление аккаунта
    - Привязка к команде по коду
Команды
    - Админ создаёт компанию/команду
    - Добавление/удаление пользователей
    - Просмотр состава команды
    - Назначение ролей
Задачи
    - Создание задач руководителем
    - Назначение исполнителя
    - Описание, дедлайн, статус
    - Изменение / удаление
    - Комментарии (упрощённый чат)
    - Статусы: открыто, в работе, выполнено
Оценка работы
    - Руководитель оценивает задачи (1-5)
    - Пользователь видит свои оценки
    - Вывод средней оценки
Встречи
    - Назначение встречи с датой/временем
    - Проверка времени (без накладок)
    - Список встреч пользователя
    - Возможность отмены
Календарь
    - Отображение задач и встреч
    - Месячный и дневной вид
    - Автоматическое добавление
Админ-панель
    - Готовая админка
    - Управление пользователями
    - Управление командами
    - Управление задачами
